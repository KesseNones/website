Python Omniprogram Project

This is a compilation of over 100 different Python programs written using the Tkinter GUI library for Python. It's called the Omniprogram because it's made of many different programs. Most of the programs are time related and some are math-related.

Some notable programs include:
  1. A metric date displayer
  2. A chinese calendar calculator
  3. A calculator that works in 36 number bases
  4. A base converter from base 1-36 to 10 and vice versa
  5. A live calendar display of the Reformed Calendar, a calendar I invented
  6. An astrological clock for those into that sort of thing
