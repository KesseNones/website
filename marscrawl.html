Mars Crawler Game Design Project

A PacMan clone built using IntelliJ IDEA and the LibGdx framework over a period of 2 weeks for a homework in my undergraduate Game Design class.

The project took a heavy amount of work, requiring practically an all-nighter to complete it at the very end. Not to mention the fact that Java is an interesting language to put it generously. The game involved implementing pathfinding for the blobs using Dijkstra's and a manual tile map system for the levels using integer level tile encodings.

The game is similar to PacMan but more of a dungeon crawler where you're a rover collecting minerals and avoiding aggressive blobs. It's a fun little game and visually kinda fun and light, if a bit unfinished.
